# phyon-ransomwere
# Simulador Educacional de Ransomware em Python

## Objetivo
Projeto desenvolvido para fins educacionais no bootcamp da DIO, com o objetivo de compreender conceitos de criptografia, segurança da informação e recuperação de arquivos em ambiente controlado.

## Tecnologias utilizadas
- Python 3

## Estrutura do projeto
- encrypter.py → realiza criptografia de arquivos de teste
- decrypter.py → realiza descriptografia
- teste.txt → arquivo usado em laboratório

## Como executar
1. Execute o script de criptografia
2. Verifique a alteração do arquivo de teste
3. Execute o script de descriptografia
4. Confirme a recuperação do conteúdo original

## Observação
Projeto criado exclusivamente para aprendizado em ambiente controlado e autorizado.
